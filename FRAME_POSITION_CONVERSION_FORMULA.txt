================================================================================
FRAME POSITION CONVERSION FORMULA
================================================================================

When positioning elements in the PosterEditorScreen, the debug output shows 
canvas coordinates. These need to be converted to reference frame coordinates 
before storing in src/data/frames.ts.

================================================================================
REFERENCE FRAME DIMENSIONS
================================================================================
Standard Width:  720 pixels
Standard Height: 487.2 pixels

These are defined in: src/utils/frameUtils.ts (lines 71-72)

================================================================================
CANVAS DIMENSIONS (typical on test device)
================================================================================
Canvas Width:  345.6 pixels (approximately)
Canvas Height: 300.93 pixels (approximately)

Note: Canvas size varies by device screen size. The debug output shows the 
actual canvas size used.

================================================================================
CONVERSION FORMULA
================================================================================

To convert from CANVAS coordinates (from debug output) to REFERENCE coordinates 
(for frames.ts):

  referenceX = canvasX √ó (720 / canvasWidth)
  referenceY = canvasY √ó (487.2 / canvasHeight)

Or using the scale factors:

  scaleX = canvasWidth / 720
  scaleY = canvasHeight / 487.2
  
  referenceX = canvasX / scaleX
  referenceY = canvasY / scaleY

================================================================================
STEP-BY-STEP PROCESS
================================================================================

1. Apply the frame in PosterEditorScreen
2. Position elements where you want them
3. Check the debug console output which shows:
   - Canvas coordinates (X, Y)
   - Canvas size (width x height)
   - Frame name (e.g., "For frame6.png, use: x: 8, y: 273")

4. Convert coordinates using the formula:
   - Get canvasWidth and canvasHeight from debug output
   - Calculate: referenceX = canvasX √ó (720 / canvasWidth)
   - Calculate: referenceY = canvasY √ó (487.2 / canvasHeight)
   - Round to nearest integer

5. Update src/data/frames.ts with the reference coordinates

================================================================================
IMPORTANT NOTES
================================================================================

1. The debug output already shows rounded coordinates suitable for direct use
   in the conversion formula.

2. Always use the CANVAS SIZE from the debug output for your calculations, as
   it may vary slightly between devices.

3. The conversion ensures that elements will appear in the exact same position
   on any device, regardless of screen size.

4. Logo and companyName positions at the top of frames typically don't need
   adjustment as they're already correctly positioned in the frame definitions.

5. After updating frames.ts, reload the app to see the changes take effect.
