================================================
TROUBLESHOOTING
================================================

ISSUE 1: Still Showing Relative Paths After Reload
--------------------------------------------------
Symptom: Console shows "/uploads/..." instead of "https://..."

Solution:
1. Stop app completely
2. Clear Metro cache: npm start -- --reset-cache
3. Delete app from device
4. Rebuild and reinstall: npx react-native run-android

ISSUE 2: URLs Are Correct But Images Still Don't Load
-----------------------------------------------------
Symptom: Console shows "https://..." but "‚ùå Image load error"

Possible causes:
A. Network/SSL Issue
   - Check device internet connection
   - Try using HTTP instead of HTTPS (testing only)
   - Check if device can access external URLs

B. Android Cleartext Traffic (HTTP)
   - If using HTTP, need to allow cleartext in AndroidManifest.xml
   - But your images are HTTPS, so this shouldn't be the issue

C. React Native Image Component Issue
   - Try using react-native-fast-image library
   - See "Alternative Solution" below

ISSUE 3: Images Load But Still White
------------------------------------
Symptom: Console shows "‚úÖ Image loaded successfully" but still white

Solution: Check styling
```typescript
// In styles, verify posterImage has:

  width: '100%',

// Verify posterCard doesn't have:
```

ALTERNATIVE SOLUTION (If standard Image fails)
================================================

If React Native's Image component has issues, use Fast Image:

Install:
```bash
npm install react-native-fast-image
cd ios && pod install && cd ..
```

Update MyBusinessScreen.tsx:
```typescript
import FastImage from 'react-native-fast-image';

// Replace <Image> with:
<FastImage
  source={{
    uri: item.thumbnail,
    priority: FastImage.priority.normal,
  }}
  style={styles.posterImage}
  resizeMode={FastImage.resizeMode.cover}
  onError={() => {
    console.error('‚ùå FastImage load error:', item.id);
  }}
  onLoad={() => {
    console.log('‚úÖ FastImage loaded:', item.id);
  }}
/>
```
