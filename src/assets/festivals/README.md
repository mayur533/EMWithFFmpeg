# Festival Images

This directory contains festival images used by the FestivalCalendar component.

## Required Images

Please add the following images to this directory:

1. **navratri.jpg** - Navratri festival image
2. **diwali.jpg** - Diwali festival image  
3. **kartik-purnima.jpg** - Kartik Purnima festival image
4. **christmas.jpg** - Christmas festival image
5. **makar-sankranti.jpg** - Makar Sankranti festival image
6. **valentine.jpg** - Valentine's Day image
7. **holi.jpg** - Holi festival image
8. **baisakhi.jpg** - Baisakhi festival image

## Image Specifications

- **Format**: JPG or PNG
- **Size**: Recommended 400x300 pixels or similar aspect ratio
- **Quality**: High quality images that represent the festival well
- **Content**: Should be appropriate and culturally respectful

## Sample Images

For development purposes, you can use placeholder images or download royalty-free images from:
- Unsplash (https://unsplash.com)
- Pexels (https://pexels.com)
- Pixabay (https://pixabay.com)

## Usage

The images are automatically loaded by the FestivalCalendar component using require() statements. Make sure the filenames match exactly as specified above.

## Fallback

If images are not available, the component will show a placeholder or handle the error gracefully.
