TROUBLESHOOTING GUIDE
================================================

PROBLEM 1: Category is "General" instead of "Decorators"
--------------------------------------------------------
CAUSE: User's business profile doesn't have category saved
FIX: Ensure registration API saves category to business profile
CHECK: Backend endpoint POST /api/installed-users/register
       Must save category field to business_profiles table

PROBLEM 2: Multiple categories in response
------------------------------------------
CAUSE: Backend API not filtering by category
FIX: Backend GET /api/mobile/posters/category/{category}
     Must filter: WHERE category = :category
CHECK: Backend query should be:
       SELECT * FROM posters WHERE category = 'Decorators'

PROBLEM 3: No posters found
---------------------------
CAUSE: No posters exist for that category in database
FIX: Add posters for the category in backend database
CHECK: Run query: SELECT * FROM posters WHERE category = 'Decorators'

PROBLEM 4: API error / failed response
--------------------------------------
CAUSE: Backend endpoint not implemented or server down
FIX: Implement backend endpoint: GET /api/mobile/posters/category/{category}
CHECK: Test endpoint directly: curl http://localhost:3001/api/mobile/posters/category/Decorators
