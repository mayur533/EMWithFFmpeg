‚ùå OPTION 2: REMOVE IT COMPLETELY
========================================

‚ö†Ô∏è Use only if you DON'T want any analytics

Step 1: Remove the permission from manifest
Add this to android/app/src/main/AndroidManifest.xml:

<manifest>
    <!-- Opt out of AD_ID permission -->
    <uses-permission 
        android:name="com.google.android.gms.permission.AD_ID"
        tools:node="remove"/>
</manifest>

Step 2: Add to manifest tag:
xmlns:tools="http://schemas.android.com/tools"

Step 3: Disable Firebase Analytics data collection
Add to android/app/src/main/AndroidManifest.xml inside <application>:

<meta-data
    android:name="google_analytics_adid_collection_enabled"
    android:value="false" />

<meta-data
    android:name="google_analytics_ssaid_collection_enabled"
    android:value="false" />

‚ö†Ô∏è WARNING: This will disable:
- User analytics
- Campaign tracking
- Performance insights
- Demographics data
